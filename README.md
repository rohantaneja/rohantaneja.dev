## Credits

This project is a Next.js rebuild of the [vCard Personal Portfolio](https://github.com/codewithsadee/vcard-personal-portfolio) template by [codewithsadee](https://github.com/codewithsadee). The original template was built with vanilla HTML, CSS, and JavaScript.

**What's different in this version:**
- Rebuilt from scratch using Next.js 16 and React 19
- Styled with Tailwind CSS
- Dark/Light theme support with system preference detection
- TypeScript for type safety
- Optimized for Vercel deployment

## Installation & Set Up

1. Install dependencies

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm run dev
   ```

## Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

2. Preview the site as it will appear once deployed

   ```sh
   npm run start
   ```
